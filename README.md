# 🚲 Bike Sharing Rental Demand Prediction

## 📌 Project Overview
This project predicts hourly bike rental demand using environmental and temporal factors such as weather, season, holiday, temperature, humidity, and working days.

The goal is to optimize bike availability and reduce waiting times by forecasting rental demand.

---

## 🎯 Business Problem
Bike sharing systems often face demand-supply imbalance. Accurate prediction helps:

- Improve bike availability
- Reduce customer waiting time
- Optimize operations
- Improve pricing strategies
- Support marketing decisions

---
